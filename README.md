## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/jayvaidya30/portfolio-v2
   cd portfolio-v2
   ```

2. Install dependencies:

   ```bash
   pnpm install
   ```

3. Start the development server:

   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000)
